About The Site5 Engineering Team

The Site5 Engineering Team is made up of highly talented individuals working to help allow Site5 customers to have an amazing hosting partner. We're full believers in the open source community and contribute ruby gems and code whenever possible. We utilize a combination of languages and technologies: Ruby, PHP, Perl, bash to name a few.

Among the various projects the Site5 Engineering Team has worked on include Synco (our internal customer resource management system) and its client-facing counterpart Backstage, SiteAdmin (our custom cPanel theme used in our shared hosting plans), a custom helpdesk, and custom staff tools for easy automation.

We ♥ open source.

We develop gems and publicize code that can be used by the masses. Among the gems we've released:

  • Lumberg - Ruby library for the WHM & cPanel API
  • Squall - Ruby library for working with the OnApp REST API
  • SolusVM - Ruby client for working with the SolusVM API
  • FannyPack - Ruby library for working with the Fantastico API
  • Hamburglar - Ruby library for interacting with the MaxMind fraud service

Visit our github account for the repositories!
